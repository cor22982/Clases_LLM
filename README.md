## Ejecucion
Dirigete a la carpeta del proyecto
```
cd Proyecto2
``` 
Activa el entorno con
```
.\proyecto2\Scripts\activate
``` 

instalar las librerias con
```
pip install -r requirements.txt
``` 
Ahora ejecuta el main usando streamlit
```
streamlit run main.py
```
